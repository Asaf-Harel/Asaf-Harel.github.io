MAMotorController

  • setvoltage(double setPower) -> void:
    Sets the voltage output of the SpeedController. Compensates for the current bus voltage to ensure that
    the desired voltage is output even if the battery voltage is below 12V - highly useful when the voltage
    outputs are "meaningful" (e.g. they come from a feedforward calculation).

    Parameters:
    setPower - The voltage to output Value should be between -1.0 and 1.0.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.setvoltage(0.2) // Set output voltage to 0.2
  • set(double setPower) -> void:
    Common interface for setting the speed of a speed controller.

    Parameters:
    setPower - The speed of the motor. Value should be between -1.0 and 1.0.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.set(0.5) // Set speed to 0.2
  • resetEncoder() -> void:
    Sets the sensor position to 0.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.resetEncoder() // Reset encoder
  • resatOnLimitF(boolean limit) -> void:
    Enables clearing the position of the feedback sensor when the forward limit switch is triggered.
    work with TalonSRX

    Parameters:
    clearPositionOnLimitF - Whether clearing is enabled, false as default
    timeoutMs - Timeout value in ms. If nonzero, function will wait for config success and report an error if it times out.
    If zero, no blocking or checking is performed.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.resatOnLimitF(true) // Clearing the position of the feedback sensor when the forward limit switch is triggered
  • resatOnLimitR(boolean limit) -> void:
    Enables clearing the position of the feedback sensor when the reverse limit switch is triggered.
    work with TalonSrx

    Parameters:
    void clearPositionOnLimitF - Whether clearing is enabled, false as default
    timeoutMs - Timeout value in ms. If nonzero, function will wait for config success and report an error if it times out.
    If zero, no blocking or checking is performed.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.resatOnLimitR(true) // Clearing the position of the feedback sensor when the reverse limit switch is triggered
  • PhaseSensor(boolean PhaseSensor) -> void:
    Sets the phase of the sensor. Use when controller forward/reverse output doesn't correlate to
    appropriate forward/reverse reading of sensor. Pick a value so that positive PercentOutput yields
    a positive change in sensor. After setting this, user can freely call SetInverted() with any value.

    Parameters:
    PhaseSensor - Indicates whether to invert the phase of the sensor.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.PhaseSensor(true) // Invert the phase of the sensor
  • getPosition() -> double:
    Get the selected motor position (in raw sensor units).

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.getPosition() // Get the motor position
  • getVelocity() -> double:
    Get the selected motor velocity.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.getVelocity() // Get the motor velocity
  • getForwardLimitSwitch() -> boolean:
    Get the value from the forward digital input.

    Retrieve the value of a single digital input channel from a motor controller. This method will
    return the state of the limit input based on the selected polarity, whether or not it is enabled.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.getForwardLimitSwitch() // Get the forward limit switch state
  • getReversLimitSwitch() -> boolean:
    Get the value from the reverse digital input. Retrieve the value of a single digital input channel
    from a motor controller. This method will return the state of the limit input based on the selected
    polarity, whether or not it is enabled.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.getReversLimitSwitch() // Get the reverse limit switch state

    Parameters:

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.overrideLimitSwitches(true) // Set limit switches to enable
  • getOutput() -> double:
    Common interface for getting the current speed of a speed controller.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.getOutput() // Get motor speed (between -1.0 to 1.0)
  • getStatorCurrent() -> double:
    Get the motor controller's output current in Amps.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.getStatorCurrent()
  • configRampRate(double rampRate) -> void:
    Sets the ramp rate for open loop control modes.

    This is the maximum rate at which the motor controller's output is allowed to change.
    Parameters:
    rampRate - Time in seconds to go from 0 to full throttle.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.configRampRate(0.8) // Set ramp rate to 0.8
  • setInverted(boolean setInverted) -> void:
    Common interface for inverting direction of a speed controller.

    Parameters:
    setInverted - The state of inversion, true is inverted.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.setInverted(true) // Invert motor
  • DisableLimit(boolean onOff) -> void:
    Disables limit switches triggering (if enabled) when the sensor is no longer detected.

    Parameters:
    onOff - Disable triggering.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.DisableLimit(true) // Disable limit switch triggering
  • changeMood(boolean onOff) -> void:
    Sets the mode for the motor controler.

    Parameters:
    onOff - Idle mode: Brake = true, Coast = false.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor.changeMood(false) // Set motor to coast
  • followIMotorController(MAMotorControler motor) -> void:
    Set the control mode and output value so that this motor controller will follow another motor controller.
    Currently supports following Victor SPX and Talon SRX.

    Parameters:
    motor - Motor controller to follow.

    Example:
    MAMotorControler motor1 = new MAMotorControler(...); // Initialize new MAMotorControler instance
    MAMotorControler motor2 = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor2.followIMotorController(motor1) // Set motor2 to follow motor1
  • followSparkMax(MAMotorControler motor) -> void:
    Causes this controller's output to mirror the provided leader.
    Only voltage output is mirrored. Settings changed on the leader do not affect the follower.
    The motor will spin in the same direction as the leader. This can be changed by passing a true
    constant after the leader parameter. Following anything other than a CAN SPARK MAX
    is not officially supported.

    Parameters:
    motor - Motor controller to follow.

    Example:
    MAMotorControler motor1 = new MAMotorControler(...); // Initialize new MAMotorControler instance
    MAMotorControler motor2 = new MAMotorControler(...); // Initialize new MAMotorControler instance
    motor2.followSparkMax(motor1) // Set motor2 to follow motor1

MAPIDController

  • setOutputRange(double low, double high) -> void:
    Set the OutPutRange of the PIDController

    Parameters:
    low - The lower boundary to which to clamp value.
    high - The higher boundary to which to clamp value.

    Example:
    MAPidController PIDController = new MAPidController(...); // Initialize new MAPidController instance
    PIDController.setOutputRange(0.25, 0.73) // Set the output range to 0.25-0.73
  • setSetpoint(double setPoint) -> void:
    Sets the distance for the PIDController.

    Parameters:
    setPoint - The desired setpoint.

    Example:
    MAPidController PIDController = new MAPidController(...); // Initialize new MAPidController instance
    PIDController.setSetpoint(120) // Set the setpoint to 120
  • atSetpoint(double waitTime) -> boolean:
    Returns true if the error is within the percentage of the total input range, determined by
    setTolerance and if the PID was at atSetPoint for a Period of time. This asssumes that the maximum
    and minimum input were set using

    Parameters:
    waitTime - The Time the PID need to be atSetPoint.

    Example:
    MAPidController PIDController = new MAPidController(...); // Initialize new MAPidController instance
    PIDController.setSetpoint(100) // Set the setpoint to 100
    PIDController.setTolerance(10) // Set the tolerance to 10
    PIDController.atSetpoint(0.2) // Return true if the robot is between 90-110 for 0.2 seconds, else return false
  • calculate(double measurement, double setPoint) -> double:
    Returns the next output of the PID controller.

    Parameters:
    measurement - The current measurement of the process variable (for example: encoder value).
    setpoint - The new setpoint of the controller.

    Example:
    MAMotorControler motor = new MAMotorControler(...); // Initialize new MAMotorControler instance
    MAPidController PIDController = new MAPidController(...); // Initialize new MAPidController instance
    PIDController.calculate(motor.getPosition(), 1000) // Calculate the next output with the encoder value and setpoint at 1000
